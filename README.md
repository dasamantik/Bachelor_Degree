## Технології

- **Frontend:**
  - Vue.js для клієнтської частини.
  - Vuex для управління станом додатка.
  - Vue Router для реалізації навігації.

- **Backend:**
  - Node.js та Express.js для серверної частини.
  - MongoDB для зберігання даних.
  - Passport.js для автентифікації.
